# Online Customer Support Ticket System

## Project Overview
The Online Customer Support Ticket System is a web-based application developed to help users raise, manage, and track customer support issues efficiently. The system provides an organized way for customers and support teams to communicate through support tickets.

This project was developed using HTML, CSS, and JavaScript for the frontend and Python for the backend.

---

## Features
- User Registration and Login
- Create and Manage Support Tickets
- Track Ticket Status (Open, In Progress, Resolved)
- Admin Dashboard for Ticket Management
- Ticket Priority and Category Management
- Responsive User Interface
- Real-time Updates and Notifications
- Search and Filter Tickets
